ThinkPHP 5.1 后台管理系统
===============

## 安装
1.安装tp5.1框架<br/>
2.在项目根目录执行composer require liupan-v/think-admin dev-develop<br/>
3.配置框架的数据库信息<br/>
4.在项目根目录下执行：php think tpadmin:install

## 使用
后台地址： /admin (请先自行配置thinkphp5的URL重写去掉index.php)<br/>
默认账号：admin <br/>
默认密码：123456 <br/>
演示地址：http://demo.115.org.cn/admin （请勿修改默认管理员密码谢谢）
